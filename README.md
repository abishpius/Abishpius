## GitHub Pages Deployment

This website is deployed using GitHub Pages, providing free hosting directly from the repository.

### Deployment Setup

1. **Repository Configuration**
   - Repository: `abishpius/Abishpius`
   - Branch: `master` (GitHub Pages source)
   - Custom domain: Not configured (uses github.io subdomain)

2. **Automatic Deployment**
   - Any push to `master` branch triggers automatic deployment
   - Changes are live within minutes
   - No build process required (static files)

3. **Access URLs**
   - Live site: `https://abishpius.github.io/Abishpius/`
   - Repository: `https://github.com/abishpius/Abishpius`

### Local Development

To run the website locally:

```bash
# Clone the repository
git clone https://github.com/abishpius/Abishpius.git
cd Abishpius

# Start local server (Python 3)
python -m http.server 8000

# Or use Python 2
python -m SimpleHTTPServer 8000

# Access at http://localhost:8000
```
